#### MODULE NAME
       SameBit

#### DEPENDENCIES
       None

#### INPUTS
Name | Size(Bits)
-----|------------
A   |     1      

#### OUTPUTS
Name | Size(Bits)
-----|------------
Y   |     1      

#### DESCRIPTION
       Takes  a  single-bit input A, and outputs the same bit into Y.  This is
       useful in Quartus when you need two different names for the same  wire,
       which is not allowed.
